# Release Checklist - Last Shift

## Gameplay Stability

- [ ] Full Level 1 loop can be completed.
- [ ] No known softlocks.
- [ ] No major collision issues.
- [ ] Ending triggers reliably.
- [ ] Packaged build matches Play in Editor behavior.

## Level Readability

- [ ] Main path is clear.
- [ ] Side rooms support the objective.
- [ ] Important interactables stand out.
- [ ] Lighting supports navigation.
- [ ] Objective feedback is understandable.

## Documentation Readiness

- [ ] README is current.
- [ ] GDD matches the implemented prototype.
- [ ] Roadmap reflects completed phases.
- [ ] QA checklist is updated.
- [ ] Asset credits are documented.
- [ ] Changelog includes release notes.

## GitHub Polish

- [ ] Repository structure is clean.
- [ ] Main branch contains stable release.
- [ ] Dev branch contains integration work.
- [ ] Issues are updated or closed.
- [ ] Screenshots are linked in README.
- [ ] Gameplay video link is added.

## Windows Build Export

- [ ] Build target is Windows.
- [ ] Build folder is named clearly.
- [ ] Build launches on a clean machine if possible.
- [ ] No editor-only content is required.
