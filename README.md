# Last Shift

Last Shift is a short first-person atmospheric horror prototype about a facility worker trapped after restoring the wrong system.

## Project Overview

Last Shift is an Unreal Engine 5 portfolio project focused on a finishable Level 1 prototype for Windows PC. The project emphasizes atmosphere, environmental storytelling, simple Blueprint interactions, and a clear short-form gameplay loop.

The player returns from a repair task to find an old industrial facility empty. To escape, they must explore offices and maintenance rooms, restore power, unlock access, and reach the control room. The final terminal appears to be the way out, but instead triggers full facility lockdown.

## Gameplay Overview

- Perspective: first-person
- Genre: atmospheric horror / narrative exploration
- Combat: none
- AI: none for Level 1
- Prototype goal: one complete playable level with a beginning, middle, and ending

## Core Gameplay Loop

Explore -> Find Fuse -> Restore Power -> Access Terminal -> Unlock Control Room -> Trigger Lockdown Ending

## Key Features

- Compact industrial horror level
- First-person exploration
- Simple interaction prompt system
- Fuse pickup and power restoration objective
- Locked door and terminal progression
- Environmental storytelling through props, notes, logs, lighting, and sound
- Lockdown ending sequence for Level 1

## How to Open the Project

1. Install Unreal Engine 5.
2. Clone or download this repository.
3. Open the `.uproject` file once it has been created.
4. Let Unreal generate any required intermediate files.

If the `.uproject` file is not present yet, create the Unreal project first and keep this repository structure.

## How to Run the Level

1. Open the project in Unreal Engine 5.
2. Open the Level 1 map from `Content/Maps`.
3. Press Play in Editor.

The planned map name is `LVL_Level01_LastShift`.
